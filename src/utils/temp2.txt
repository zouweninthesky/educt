Открываем конкретный степ. в currentStep помещается весь объект этого степа. 

currentOption - это комментарий. 
Когда пользователь открывает комментарий - в currentOption помещается старое значение комментария, в поле для ввода помещается старое значение комментария. 
Если после этого пользователь нажимает "Сохранить" или "Отмена" - значение из поля комментария сравнивается с currentOption, 1) если оно такое же - ничего не 
происходит, currentOption становится null 2) если значение отличается - на "Отмене" вылезает предупреждение, что изменения не сохранятся, если пользователь 
подтверждает - currentOption становится null - на "Сохранить" значение комментария заменяется на значение поля комментария в currentStep, а затем проверяется, есть ли 
степ с таким uid в списке modified. Если есть - он целиком заменяется на значение currentStep, если нет - currentStep пушится в modified. currentOption 
становится null.

действия. Если после этого пользователь нажимает "Сохранить" или "Отмена" - значения из стилей элемента действия сравниваются с теми, которые сохранены в 
currentOption. , 1) если они такие же - ничего не происходит, currentOption становится null 2) если значения отличаются - на "Отмене" вылезает предупреждение,
что изменения не сохранятся, если пользователь подтверждает - currentOption становится null - на "Сохранить" значение действия заменяется на currentOption в 
currentStep, а затем проверяется, есть ли степ с таким uid в списке modified. Если есть - он целиком заменяется на значение currentStep, если нет - 
currentStep пушится в modified. currentOption становится null.
